# eliteMorseDraw

This is an updated fork of eliteMorseDraw that was previously published by "unwantedusername".

This fork was created as the original project had not been updated since late 2016, as such it did not include the ships from "Horizons" (2.0), "Beyond" (3.0) or "Odyssey" (4.0).

Unfortunately, the original project is no longer available, and was previously at: `https://github.com/unwantedusername/eliteMorseDraw`
